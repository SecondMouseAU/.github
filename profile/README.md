# SecondMouse

Building a native **Swift + OpenCASCADE** CAD stack for Apple platforms — and the tools and
products on top of it.

## OCCTSwift

**OCCTSwift** is an open ecosystem of Swift packages wrapping OpenCASCADE Technology (OCCT)
for B-Rep solid modelling on macOS and iPad: the geometry kernel, meshing, file I/O
(STEP / IGES / STL), an interactive Metal viewport, and a CLI / MCP toolchain.

Explore the public repositories on this org's [Repositories](https://github.com/orgs/SecondMouseAU/repositories) tab.
