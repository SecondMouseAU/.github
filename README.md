# .github

Org-wide defaults for **SecondMouseAU**.

- `profile/README.md` — the public org profile page.
- `.github/ISSUE_TEMPLATE/` — shared issue forms inherited by every repo without its own.
- `labels.yml` — the canonical label taxonomy (synced to repos via the label-sync workflow).

Cross-repo coordination, roadmap, and the knowledge catalog live in the private
[`ecosystem`](https://github.com/SecondMouseAU/ecosystem) repo.
